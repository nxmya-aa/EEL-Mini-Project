## How to run
1. Unzip the package.
2. Open `index.html` in your browser (double-click). For best results (some browsers block local files), run a local server:
   - `python -m http.server 8000`
   - then open http://localhost:8000

## Notes
- All data on the dashboard is simulated for demo.
- Replace team placeholders and image with your own assets before submission.
